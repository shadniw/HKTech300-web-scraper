# HKTech300 Web Scraper

## Overview
A Python web scraper that extracts start-up company information from the 
CityU HKTech300 programme website across 40 pages.

## Data Collected
- Company Name
- CityU URL
- Company Website
- Email

## Tools Used
- Python
- Selenium
- BeautifulSoup
- Pandas

## Output
720 companies saved to hktech300_startups.xlsx
